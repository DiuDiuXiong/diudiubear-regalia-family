Diu Diu Bear's Regalia Arsenal — v1.0.0

CONTENTS
- Regalia: a modular 7.62x51 battle rifle.
- Regalia Prism: a faster, stronger colourful variant.
- Exclusive stocks, grips, muzzle devices, magazine, and hidden barrel core.
- Custom firing, suppressor, and reload sounds.
- Diu Diu Bear, a bunker trader located beside the Networker.

INSTALLATION
Place the complete "diudiubear_regalia_family" folder in ZERO Sievert's local mods directory, enable it in the Mod Manager, and reload mod data.

WORKSHOP
This package is prepared for a first Steam Workshop upload. It intentionally contains no workshop_id. After the first upload, add the new Workshop ID to meta.ini for future updates.

COMPATIBILITY
All custom internal IDs use the wz_ddb_ namespace to reduce conflicts with other mods. Do not enable both a local copy and a subscribed Workshop copy at the same time.
